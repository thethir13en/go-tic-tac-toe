# Implementation of Tic-Tac-Toe game written on Golang, console-variant

## Install

- just run:

    - Linux:
        ```
        go build -o $GOPATH/bin/go-tic-tac-toe
        ```
    - Windows:
        ```
        go build -o $Env:GOPATH\bin\go-tic-tac-toe.exe
        ```

- launch executable in your ```$GOPATH/bin``` directory
